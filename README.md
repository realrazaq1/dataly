# What is Dataly

Dataly is a data vending application where customers can buy data subscription for their mobile phones. 

## Live Preview
https://dataly.herokuapp.com/

## Technologies used 
- NodeJS 
- MongoDB 
- Express 
- Mongoose 
- EJS 
- Paystack API & Library 
## Application type 
Fullstack

## Payment Gateway used 
Paystack 

### Tags 
- data vending  
- ecommerce 
- payment integration 
- API
